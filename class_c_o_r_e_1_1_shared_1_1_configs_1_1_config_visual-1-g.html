Detailed Description

If you have a text field of the Enum type in your config, but you don't know whether this parameter is in your Enum list, you can create this class. It will have 2 parameters: a text value and an Enum. If there is no Enum, use a text one. This class was designed to work with mods. After all, modders may not know your internal rules.
