Detailed Description

Static class holding model data for Asset Bundle Browser tool. Data in Model is read from DataSource, but is not pushed.

If not using a custom DataSource, then the data comes from the AssetDatabase. If you wish to alter the data from code, you should just push changes to the AssetDatabase then tell the Model to Rebuild(). If needed, you can also loop over Update() until it returns true to force all sub-items to refresh.

ABDataSource CORE.AssetBundleBrowser.AssetBundleModel.Model.DataSource
staticgetset

If using a custom source of asset bundles, you can implement your own ABDataSource and set it here as the active DataSource. This will allow you to use the Browser with data that you provide.

If no custom DataSource is provided, then the Browser will create one that feeds off of and into the AssetDatabase.
